# Prime_Number_Generator

Here I have built Web Application which will serve prime numbers as per the input given in the range. It is based on Python,flask framework.

## Requirements
- Python 3.5+
- Flask 1.1.2
- jinja2 2.11.2
- Html 

###  Step 1:

- Clone the repository.
- ```git clone https://github.com/sudharsudhar/Prime_Number_Generator.git```

###  Step 2:
Install the Requirements
###  Step 3:
[ In Command prompt or in any IDE Terminal ]

```bash
$ cd Prime_Number_Generator


# To Install all depenices under on environment
# create Virtual environment and install  all dependices.
$ virtualenv vir_env_name
$ virtualenv -p /usr/bin/python3 vir_env_name
$ source vir_env_name/bin/activate

$ pip install -r requirements.txt
```
### Note: Change the path in Line No:8
(i.e) Mention the path were you want to store the DB

Then run the below cmd:
```
$ python app.py
```
